Punto 12
--------
Pharo tiene una ventana llamada “Inspector” que permite ver y cambiar las variables de instancia de un objeto. Evalúe este código y diga que observa:

|a|
a := #( 1 2 sam 'joe' ( 4 5 ) ).
a at: 2 put: 3 / 4.
a inspect

Resultado
---------
Inspector on an Array [5 items] {1. (3/4). #sam. 'joe'. #(4 5)}:

    Index    Item
    -----    ----
    1        1
    2        (3/4)
    3        #sam
    4        'joe'
    5        an Array [2 items] (4 5)
